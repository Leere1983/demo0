# Spring Boot 项目

这是一个用于自我学习的 Spring Boot 项目

项目要求如下：

自行搭建个springboot项目，不要求复杂度，但需要用到下面各知识点:

1. 学习springboot相关内容:MVC、依赖注入、事务、AOP、拦截器
2. 数据库:基本 SQL、索引、事务基本概念
3. redis:核心概念、redis数据结构、分布式锁
4. rocketmg:核心概念的了解、生产者消费者的基本使用

# 环境要求

- **Java 17** 
- **Maven 3.9.9**
- **MySQL 8.0** 
- **Redis 5.0.14.1** 
- **RocketMQ 5.1.0**
